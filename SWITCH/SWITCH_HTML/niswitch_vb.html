在 Visual Basic 中使用 NI-SWITCH

本主题假定您使用 Microsoft Visual Basic ADE 来管理您的代码开发并且您熟悉 ADE。

要在 Visual Basic 中开发 NI-SWITCH 应用程序，请遵循以下一般步骤：

  1. 打开现有的或新的 Visual Basic 项目。
  2. 创建应用程序所需的文件并将它们添加到项目中。
  3. 添加对作为 NI-SWITCH DLL 一部分的 National Instruments Switch Library (NISWITCH) 的引用。在 Visual Basic 6.0 中，选择Project»References菜单选项和 National Instruments Switch Library (NISWITCH)。如果您没有在此处看到 NISWITCH，请单击浏览并在您的IVI\Bin目录中找到NISWITCH_32.dll 。
  4. 使用对象浏览器 <F2> 查找函数原型和常量。
  5. 将 NI-SWITCH 函数调用添加到您的应用程序。
  6. 单击Run运行您的应用程序。

示例程序

要使用 Visual Basic 6.0 或更高版本加载示例项目，请选择文件»打开项目，然后选择您选择的.vbp文件。

有关 NI-SWITCH 示例的更多信息，包括示例位置，请参阅NI-SWITCH 编程示例。

特别注意事项

字符串传递

在 Visual Basic 中，数据类型 String 的变量不需要特殊修改即可传递给 NI-SWITCH 函数。Visual Basic 会在将字符串（通过引用，因为在 Visual Basic 中不能通过值传递）传递给过程或函数之前自动将空字符附加到字符串的末尾。

参数传递

默认情况下，Visual Basic 通过引用传递参数。如果需要按值传递，请在前面添加 ByVal 关键字。
