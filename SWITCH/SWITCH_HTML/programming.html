使用 NI-SWITCH 进行编程

要对您的开关模块进行编程，请选择一个 API——NI-SWITCH 或 NI-DAQmx。有关这些 API 的更多信息，请参阅NI Switches Getting Started Guide。

使用 NI-SWITCH API

本书涵盖了使用 NI-SWITCH API 对开关模块进行编程。涵盖以下主题：

  • 入门——如何开始在 LabVIEW、LabWindows/CVI、Visual C++ 和 Visual Basic 中创建应用程序
  • 编程流程- 即时操作和扫描的编程流程
  • 功能- 使用交换机模块及其功能的基于任务的信息
  • 示例— NI LabVIEW、LabWindows/CVI、Visual C++ 和 Visual Basic 中的示例
  • 错误和状态代码— NI-SWITCH 和 IviSwtch 的错误代码

使用 NI-DAQmx API

要使用 NI-DAQmx API 对开关模块进行编程，请参阅NI-DAQmx 帮助。
