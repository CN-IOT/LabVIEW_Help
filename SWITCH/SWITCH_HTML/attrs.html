设置和检查属性和属性

属性和属性分别允许访问 LabVIEW 和 CVI\C\C++\VB 的高级配置选项和信息。

NI-SWITCH 包含设置大多数开关模块属性/属性的高级 VI/函数。尽可能多地使用高级驱动程序 VI/函数，因为它们处理顺序依赖和多线程锁定。

特性

在 LabVIEW 中，您可以使用NI-SWITCH 属性节点获取（读取）或设置（写入）属性。有关详细信息，请参阅 niSwitch 属性节点。

NI-SWITCH 属性的完整列表请参考niSwitch 属性。

属性

在 C 和 Visual Basic 中，使用niSwitch_SetAttribute和niSwitch_GetAttribute函数访问属性。这些函数对应于特定的数据类型。例如，要设置NISWITCH_ATTR_CONTINUOUS_SCAN（数据类型 ViBoolean），请使用niSwitch_SetAttributeViBoolean函数。

有关NI-SWITCH 属性的完整列表，请参阅 NI-SWITCH 属性。
