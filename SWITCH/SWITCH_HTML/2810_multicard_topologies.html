NI 2810 Multicard Topologies

The following table lists possible topologies for an NI 2810 NI SwitchBlock device. Refer to Initializing with Topology for NI SwitchBlock Devices for information about using the Configured Topology and NISWITCH_TOPOLOGY_CONFIGURED_TOPOLOGY software names.

Note  You must connect pin 96 (INTERLOCK ENABLE) to pin 48 (GND) with a 500 Ω resistor (500 Ω ±20%, 1/10 W minimum) to allow connection to the analog bus. Refer to the NI SwitchBlock Safety Interlock topic for additional information about this safety feature.
