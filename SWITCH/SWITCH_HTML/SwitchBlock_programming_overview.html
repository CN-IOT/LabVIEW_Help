NI SwitchBlock 概述

NI SwitchBlock 是专为大型矩阵测试系统设计的开关平台。它包括内部列扩展功能，可以在单个 PXI 机箱中创建具有 8,800 多个交叉点的矩阵。为了提高系统的长期可靠性，NI SwitchBlock 配备了新的 NI Switch Health Center，其中包含集成的继电器测试、继电器计数跟踪和报告生成工具。

尽管 National Instruments 建议使用 NI Switch Executive 对 NI SwitchBlock 进行编程，但您可以使用 NI-SWITCH 和 NI-DAQmx 驱动程序 API 对 NI SwitchBlock 进行编程。NI Switch Executive 智能路由管理软件支持大型矩阵应用程序编程，并包含一个配置向导，可通过 NI SwitchBlock 加速开发。本帮助主题介绍了如何使用 NI-SWITCH 和 NI-DAQmx 驱动程序 API 对 NI SwitchBlock 进行编程。
