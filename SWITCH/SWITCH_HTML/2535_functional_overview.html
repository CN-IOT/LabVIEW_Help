NI PXI-2535

The NI PXI-2535 is a 544-crosspoint, high-density matrix switch module for the PXI platform. The NI PXI-2535 is composed of FET switches. The PXI-2535 has additional fault protection circuitry to prevent overvoltage conditions. If proper grounding is not used on certain applications, relays will appear stuck open. For more information about FET fault protection circuitry, visit ni.com/info and enter the Info Code exfcjv.
