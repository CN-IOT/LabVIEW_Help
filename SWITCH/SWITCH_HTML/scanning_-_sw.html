编写软件触发扫描程序

要编写软件触发扫描程序，请完成以下步骤：

  1. 使用niSwitch Initialize With Topology VI 或niSwitch_InitWithTopology函数设置开关模块的 拓扑。
  2. 使用niSwitch 配置扫描触发VI 或niSwitch_ConfigureScanTrigger函数将触发输入( triggerInput ) 参数设置为软件触发。
  3. 使用niSwitch Set Continuous Scan VI 或niSwitch_SetContinuousScan函数设置开关在扫描列表中循环的次数，一次或无限次。
  4. 使用niSwitch 配置扫描列表VI 或niSwitch_ConfigureScanList函数设置连接列表。有关语法信息，请参阅扫描列表。
  5. 使用niSwitch Initiate Scan VI 或niSwitch_InitiateScan函数启动扫描。扫描列表中的第一个条目被执行，并且交换机等待软件触发来执行列表中的后续条目。
  6. 通过调用niSwitch Send Software Trigger VI 和niSwitch_SendSoftwareTrigger函数执行扫描列表中的每组连接。
  7. 使用niSwitch Abort Scan VI 或niSwitch_AbortScan函数终止扫描操作。
  8. 使用niSwitch Close VI 或niSwitch_close函数释放资源。

有关基本软件扫描应用程序的说明，请参阅niSwitch 软件扫描示例。

注意  某些设备可能不支持扫描。有关支持的功能的更多信息，请参阅您的设备手册。
