NI-SWITCH VI

2019 年 2 月

使用 NI-SWITCH 选板上的 VI 构建程序框图。

调色板对象 描述
niSwitch用拓扑初始化VI

返回一个会话句柄，用于在所有后续仪器驱动程序调用中识别开关模块并设置开关模块的拓扑。
niSwitch 关闭 VI

终止 NI-SWITCH 会话及其所有属性，并释放驱动程序使用的所有内存资源。
niSwitch 连接通道 VI

在通道之间建立连接。

niSwitch 断开通道 VI

断开通道之间的连接。

niSwitch 等待去抖动 VI

暂停，直到所有创建的路径都已解决。在niSwitch Get Relay Count VI 之前调用该 VI 以确保准确的继电器计数。
niSwitch 设置路径 VI

通过在路径列表参数中指定显式路径来连接两个通道。该 VI 在路径可重复性很重要的情况下非常有用，例如在校准的信号路径中。如果不需要路径可重复性，请使用niSwitch Connect Channels VI。

要获取给定连接的准确路径，请使用niSwitch Get Path VI。
niSwitch 可以连接通道 VI

验证是否可以创建通道 1和通道 2之间的路径。

如果交换机模块中可能存在路径，则在现有连接的情况下返回该路径的可用性。如果路径可能但正在使用，则返回隐式连接存在警告。
niSwitch 断开所有通道 VI

破坏所有现有路径。

如果交换模块不能中断所有路径，则返回警告。如果参数中的错误描述了调用该VI之前发生的错误，NI-SWITCH仍会尝试断开所有连接并返回从error in传入的错误信息。

使用niSwitch Relay Control VI关闭的继电器也会断开。
niSwitch 开关去抖 VI

通过返回Is Debounced属性的值指示是否所有创建的路径都已解决。
niSwitch 获取路径 VI

返回一个字符串，该字符串标识使用niSwitch Connect Channels VI创建的显式路径。将此字符串传递给niSwitch Set Path VI，以便在以后的连接中建立完全相同的路径。


子调色板 描述
扫描

使用 NI-SWITCH 扫描 VI 配置 NI 开关的扫描。

中继

使用 NI-SWITCH 继电器 VI 来控制和查询 NI 开关的各个继电器。

效用

使用 NI-SWITCH 实用程序 VI 可以使用 NI 开关的附加功能。
