niSwitch用拓扑初始化VI

返回一个会话句柄，用于在所有后续仪器驱动程序调用中识别开关模块并设置开关模块的拓扑。

该VI为资源名称参数中指定的开关模块创建一个新的IVI仪器驱动会话。NI-SWITCH 使用拓扑名称参数中指定的拓扑并覆盖 MAX 中指定的拓扑。

默认情况下，开关模块复位为已知状态。

通过指定拓扑并将模拟参数设置为 True 来启用模拟。

拓扑名称指定用于资源名称中指定的交换机模块的拓扑。为拓扑名称创建一个控件以从拓扑列表中进行选择。

reset device指定初始化时是否复位开关模块。默认值为 TRUE (1)。

注意重新启动计算机后  第一次调用niSwitch Initialize With Topology VI将重置硬件。这是不支持重置标志的唯一情况。
对 (1)

设备将重置。
错误 (0)

设备不会重置。

error in描述在此节点运行之前发生的错误情况。该输入提供了功能上的标准错误。

如果在该 VI 运行之前发生错误，则状态为 TRUE (X)，或者为 FALSE（复选标记）以指示警告或在该 VI 运行之前未发生错误。默认值为假。

code是错误或警告代码。默认值为 0。如果状态为 TRUE，则代码为非零错误代码。如果状态为 FALSE，则代码为 0 或警告代码。

source描述了错误或警告的来源，在大多数情况下，它是产生错误或警告的 VI 的名称。默认值为空字符串。

simulate enables simulation of the switch module specified in resource name. The default value is FALSE (0).

TRUE (1)

The module will be simulated.
FALSE (0)

The module will not be simulated.
